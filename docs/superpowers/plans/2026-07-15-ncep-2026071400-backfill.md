# NCEP 2026071400Z Backfill Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 以現有 NCEP adapters 與 DataStore 一次性回補 GEFS、AIGEFS、AIGFS 的 `2026071400Z` 靜態 cycle，讓既有前端日期下拉選單可直接選用。

**Architecture:** 先在不寫檔的階段下載並驗證三個來源都精確回傳 `2026071400`；全部成功後才以實際發布時間逐一交給 DataStore。這會保留 manifest 現有最新 cycle 與狀態，並將歷史 cycle 依時間排序加入既有最多 12 筆清單。

**Tech Stack:** uv、Python 3.12+、httpx、Pydantic models、現有 cyclone_tracker adapters／DataStore、靜態 JSON。

## Global Constraints

- 只回補 `gefs`、`aigefs`、`aigfs` 的 exact cycle `2026071400`。
- 三個來源必須先全部成功下載、解析且含 `09W`、`97W`，之後才可寫入任何檔案。
- 不使用 `update_sources(now=歷史時間)`，避免 manifest `generated_at` 倒退。
- 不新增 CLI `--cycle`、GitHub Actions input、Vercel 日期 API、HGEFS 或永久保存例外。
- 不重抓或改寫 ECMWF cycle。
- 回補資料沿用每來源最近 12 個成功 cycle 的保存規則。
- 寫入後必須通過完整資料樹驗證；提交只包含三個 cycle JSON 與 manifest。

## File Structure

- `public/data/gefs/2026071400.json`：GEFS `09W`／`97W` 歷史 ensemble cycle。
- `public/data/aigefs/2026071400.json`：AIGEFS `09W`／`97W` 歷史 ensemble cycle。
- `public/data/aigfs/2026071400.json`：AIGFS `09W`／`97W` deterministic cycle。
- `public/data/manifest.json`：加入上述 cycle 摘要，保留目前較新 cycle 在最前面。
- 不建立新的 production Python 模組或 script。
